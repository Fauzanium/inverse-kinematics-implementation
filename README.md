# inverse-kinematics-implementation

## Overview

![Inverse Kinematics](https://www.mathworks.com/discovery/inverse-kinematics/_jcr_content/mainParsys/image.adapt.full.medium.jpg/1747304447507.jpg)

(Gambar 1) Sumber: https://www.mathworks.com/discovery/inverse-kinematics.html

Inverse Kinematics adalah proses mencari Joint Angles End Effector yang diketahui, sedangkan Forward Kinematics sebaliknya, proses mencari End Effector dari Joint Angles yang ketahui.

Contoh penerapan inverse kinematics:
- Robotic Arm (Gambar 1)
- Ball Balancing Robot (https://www.youtube.com/watch?v=-1pX518wlu8&pp=ygUiYmFsYW5jaW5nIHJvYm90IGludmVyc2Uga2luZW1hdGljcw%3D%3D)
- Walking Humanoid Robot & Applied Physics
- Spider Bot
- dan lain-lain.

Untuk membuat simulasi inverse kinematics, kita bisa menggunakan library [Pinocchio]()
